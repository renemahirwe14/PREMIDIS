# PREMIDIS HR Platform

## Overview
HR management web application (PWA) for PREMIDIs SARL - large international company with ~10,000 employees across 6 countries.

## Stack
- Backend: FastAPI + MongoDB
- Frontend: React + TailwindCSS + Shadcn/UI
- Auth: JWT-based with 4 roles (super_admin, admin, secretary, employee)

## Modules
1. Employee Management (CRUD + documents)
2. Leave Management (request/approve/reject + calendar)
3. Payroll (payslip management)
4. Communication (chat + announcements)
5. Internal Rules
6. Performance (evaluations + objectives)
7. Voice AI Assistant (Whisper + TTS)
8. Attendance Tracking
9. Notifications

## Languages
FR, EN, SW, HI
